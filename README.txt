Arnav Magadala Portfolio Website

How to add photos:
1. Put image files inside assets/images/.
2. Open the related HTML file and search for "ADD PHOTO" or "ADD PROVE/CPSS/COURSEWORK COVER PHOTO HERE".
3. Replace the placeholder <div class="photo-slot"> with an image tag, for example:
   <img src="../assets/images/prove-wheel-spacer-final.jpg" alt="Final aluminum wheel spacer" />

For index.html cover images, use:
   <img src="assets/images/prove-cover.jpg" alt="Project Mila vehicle" />
